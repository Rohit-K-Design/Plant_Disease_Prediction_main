# Plant Disease Prediction 🌱

## About / Project Brief

The **Plant Disease Prediction** project is a web-based application that helps farmers, gardeners, and agricultural enthusiasts quickly identify plant diseases using images of leaves. By leveraging machine learning models trained on various plant disease datasets, the system can detect diseases accurately and provide suggestions for treatment or preventive measures.

The project consists of a **backend** API for processing images and running the machine learning model, and a **frontend** web interface where users can upload leaf images, view results, and access recommendations. This tool aims to make plant disease management easier, faster, and accessible to anyone with a smartphone or computer.

---

## 🚀 Features

- Upload plant leaf images and detect diseases  
- Provide treatment and product recommendations  
- User-friendly web interface  
- Supports multiple plants and disease types  

---

## 🛠 Technologies Used

- **Backend:** Python, Flask/FastAPI  
- **Frontend:** HTML, CSS, JavaScript, React  
- **Machine Learning:** TensorFlow / PyTorch / Scikit-learn  
- **Version Control:** Git & GitHub
